# express-es6
Configured Express to support ES6

### Clone Repo
```sh
    git clone https://github.com/saravanadev/express-es6.git
```

### Install Node Modules

```sh
   cd express-es6
   npm install
```

### Start Dev Server
```sh
   npm run dev
```

### Create Production  Build
```sh
   npm run build
```

### Clean Production Build
```sh
   npm run clean
```

### Watch Changes
```sh
   npm run nodemon
```
